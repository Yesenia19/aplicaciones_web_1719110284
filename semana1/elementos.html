< !DOCTYPE html >:El Doctype o “Declaración del tipo de documento” es una instrucción que permite al navegador que tipo de version estamos utilizando.

< html lang='es' >:Es la raíz de un documento HTML o XHTML. Tiene como parametro lang, que se utiliza para identificar el idioma del contenido.

< head >:Es donde van toda información acerca de la página pero no se muestra en la pantalla

<h2> < title >:Es el titulo de la página el cual debe ser llamativo, por lo que esta etiqueta debe estar incluida si ó si </h2>

< base >: Sirve para indicar la URL principal en caso de que la url se relacione con otra dentro de la página web

< link >: Podemos indicar con cual, el documento html se relaciona, ya sea otro archivo o recurso externo

< style >: Sirve para incluir estilos CSS que permiten incluir de colores, bordes, imágenes de fondo, entre otros, a los elementos de la página

< script >: Sirven para incluir código en lenguajes de script, los cuales el navegador es muy facil de interpretar

< meta >: Sirven para incluir información acerca de las caracteristicas de la página, y de esta forma se tengan palabras clave que ayuden al navegador a encontrar tu página

< body >:Es donde van toda información, enlaces, imagenes, etc. de la pagina

< body bgcolor >:Podemos definir el color de fondo de la página

< body background >:Podemos definir una imagen de internet de fondo de la página

< body text >:Podemos definir el color la tipografia de nuestra página web

< body leftmargin topmargin rightmargin bottommargin >:leftmargin es para definir el margen del lado izquierdo, topmargin para el margen de la parte de arriba, rightmargin para el margen del lado derecho y bottommargin para el margen de la parte de abajo.

< body link alink vlink >:Podemos definir el color del link con la etique link, el color del link cuando esta activo con la etiqueta alink y el color del link cuando ya se ha visitado con la etiqueta vlink

< h1 >:La usamos para indicar el título en nuestra página

< h2 >:La usamos para poner suntitulos en nuestra página

< h3 >:Se usa para dividir el contenido que se está tratando en esa subsección, ademas de buscar palabras clave derivadas

< p >:La usamos para dividir en parrafos un texto
